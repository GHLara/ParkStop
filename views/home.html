Sobre o nosso trabalho

Nosso trabalho é uma implementação de um sistema de estacionamento. Iremos trabalhar com dois sensores, o primeiro sensor é o Sensor de Distância/Ultrassônico que vai identificar se existe um veículo ou não na vaga, também teremos o Sensor de luminosidade que irá acender toda vez que a vaga for ocupada. Nosso sistema terá um controle de usuários e também das vagas que estão disponíveis e que não estão, o cliente também vai poder reservar sua vaga diretamente do site.

Framework escolhido

React O react é o Framework que utilizaremos no projeto para a criação de interface de usuários. Ele oferece um sistema de gerenciamento de estado para simplificar a manipulação dos dados. Decidimos usá-lo pois: - oferece muitos recursos e ferramentas úteis para facilitar o desenvolvimento web - usado por grandes empresas, sendo uma opção bem confiável - Alta performance - permite a criação de componentes reutilizáveis para serem usados novamente em outras partes da aplicação, o que reduz a quantidade de código e acelera o processo de desenvolvimento
